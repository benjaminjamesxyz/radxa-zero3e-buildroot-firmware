# Hello World Examples

This directory contains simple "Hello, World!" programs written in various programming languages.

## C
- **File:** `c/hello.c`
- **Description:** A classic "Hello, World!" program in C.
- **How to compile and run:**
  ```bash
  gcc c/hello.c -o c/hello
  ./c/hello
  ```

## C++
- **File:** `cpp/hello.cpp`
- **Description:** A simple "Hello, World!" program in C++.
- **How to compile and run:**
  ```bash
  g++ cpp/hello.cpp -o cpp/hello
  ./cpp/hello
  ```

## Rust
- **File:** `rust/main.rs`
- **Description:** A "Hello, World!" program written in Rust.
- **How to compile and run:**
  ```bash
  rustc rust/main.rs
  ./main
  ```

## Go
- **File:** `go/main.go`
- **Description:** A "Hello, World!" program in Go.
- **How to compile and run:**
  ```bash
  go run go/main.go
  # Or compile and then run:
  # go build -o go/hello go/main.go
  # ./go/hello
  ```

## Python
- **File:** `python/hello.py`
- **Description:** A basic "Hello, World!" script in Python.
- **How to run:**
  ```bash
  python3 python/hello.py
  ```
